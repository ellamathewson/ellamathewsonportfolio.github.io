Coffee Writings

API Web Project

This website stores people's cafe / drink information in order to help people remember what drinks the user liked at a particular cafe. As someone who doesn't drink a lot of coffee and forgets a lot of the jargon, it helps out so you always know what drink you like.

A second feature of the website is to learn about different drinks and what is in them. Under the 'All Drinks' tab users can scroll or filter the results from the API to read information and make a decision to try something new. Currently all the cafe locations are places around Rochester and the cost may differ from place to place but all the infromation is correct.
